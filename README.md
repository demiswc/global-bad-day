# Global Bad Day Index

An open source tool that correlates GDP shifts, news sentiment, stock market drops, and weather anomalies to map moments in time when everything went wrong simultaneously — rendered as an interactive D3 world map.

## Features
- GDP shifts
- News sentiment
- Stock market drops
- Weather anomalies

## Getting Started
To get the project up and running, use Docker Compose:
```bash
docker-compose up --build
```
